Pacotes necessários para o desenvolvimento
==================================================

1. Banco de dados MySql

apt-get install mysql-server-5.6


2. Instalar Django (versão mais recente)

pip3 install Django==1.7.3~


3. Suporte ao banco MySql no Python

pip3 install pymysql

4. Suporte ao banco PostgreSQL

apt-get install python3-psycopg2

5. Suporte ao sqlite3 (Em Ubuntu já vem instalado, em Debian não)
apt-get install sqlite3


apt-get install python3-psycopg2
